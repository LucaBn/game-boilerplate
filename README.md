# Game Boilerplate

A starter template for building web-based management or strategy games. This boilerplate provides a scalable, modern architecture to kickstart your game development with React and TypeScript.

## Tech Stack

This project uses a modern, performance-oriented stack:

- **React 18**
- **TypeScript**
- **Vite**
- **React Router**
- **Mantine** (UI components)
- **MUI (Material UI + Charts)**
- **Emotion** (CSS-in-JS)
- **i18next & react-i18next** (internationalization)
- **Sass** (optional styling)

## Features

- Modular, scalable architecture
- Component-based UI ready for game interfaces
- Multilingual support with automatic language detection
- Client-side routing for seamless navigation
- Ready-to-extend foundation for strategy, management, or RPG games

## Getting Started

### Prerequisites

- Node.js ≥ 18  
  (developed with Node 18.19.1)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/game-boilerplate.git
```

2. Navigate to the project folder:

```bash
cd game-boilerplate
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

Open your browser at http://localhost:5173 to see the app in action.

### Scripts

- `npm run dev` – Start the development server
- `npm run build` – Build the app for production
- `npm run preview` – Preview the production build
- `npm run lint` – Lint the codebase
